### 🔧 技术细节

#### 图片URL格式
```dart
// Pexels图片
'https://images.pexels.com/photos/{photo-id}/pexels-photo-{photo-id}.jpeg?auto=compress&cs=tinysrgb&w=800'

// Pravatar头像
'https://i.pravatar.cc/150?img={1-80}'
```

#### 数据结构
```dart
Post(
  id: '唯一ID',
  user: '用户名',
  avatar: '头像URL',
  location: '地点',
  content: '内容描述',
  image: '图片URL',
  likes: 点赞数,
  comments: 评论数,
  isLiked: 是否已点赞,
)
```
